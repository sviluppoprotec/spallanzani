Rapporto Interventi Cumulativo

 

Selezionando la voce di menu Rapporto Interventi Cumulativo, l’utente visualizza la maschera Rapporto Interventi Cumulativo che consente di visualizzare, attraverso vari parametri di ricerca, tutte le Rdl corrispondenti ai parametri di ricerca inseriti.

 

 

La ricerca può essere effettuata tramite i seguenti campi:

 

·         Codice Edificio: conoscendo esattamente il codice dell’Edificio/Complesso, o parte di esso, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una maschera che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “FSL” e selezionando il match code verranno presentati tutti gli edifici contenenti “FSL”. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

 

·         Ricerca: è un campo di ricerca libera che consente di fare una ricerca per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  per aprire finestra con la lista degli  Edificio.

Il sistema visualizza, se ci sono, tutti gli edifici che contengono tale parola chiave. Dopo aver compilato il campo

               fare click sul pulsante  per aprire la finestra con la lista dei Codici Edificio.

               Il sistema visualizza, se ci sono, tutti gli Edifici che contengono tale parola chiave. Dopo aver selezionato il         

Codice dell’Edificio il sistema valorizza automaticamente i seguenti campi:

·          Codice Edificio,

·          Ricerca.

·          Denominazione,

·          Indirizzo,

·          Comune,

·          Telefono,

·          Ditta,

·          Centro di Costo.

 

 

 

·         Richiesta di Lavoro: è un campo di ricerca libera in cui inserire il numero della Richiesta di Lavoro da   

        ricercare.

·         Addetto: è un campo di ricerca libera dove inserire il nominativo dell’Addetto. Se si conosce esattamente il   nominativo dell’addetto lo si può inserire direttamente  altrimenti si può inserire una parte di esso e selezionare il match code  e si apre una maschera contente tutti gli addetti che contengono il valore inserito.

       Se invece si vuole visualizzare la lista completa degli addetti, occorre lasciare il campo vuoto e fare click direttamente sul match code .

·         Data Da: è un campo di ricerca dove inserire la data da che si riferisce alla data di creazione della RdL. Per inserire una data bisogna selezionare il pulsante  che ci aprirà una finestra contenete il calendario, selezionando un giorno in automatico verrà riportato sul campo Data Da.

·         Data A: è un campo di ricerca dove inserire la data a che si riferisce alla data di creazione della RdL. Per inserire una data bisogna selezionare il pulsante  che ci aprirà una finestra contenete il calendario, selezionando un giorno in automatico verrà riportato sul campo Data Da.

·         Ordine di Lavoro: è un campo di ricerca libera in cui inserire il numero dell’Ordine di Lavoro da  

        ricercare.

·         Stato Richiesta: è un campo con menu a tendina in cui selezionare lo Stato Richiesta (Richiesta in attesa di emissione, Attività Completata, Emessa in Lavorazione, etc.).

     ·         Servizio: è un campo con menu a tendina in cui selezionare il Servizio (E - Montaggi, E - Infissi Interni-Esterni, 

              E - Coperture, etc.).

      ·         Gruppo: è un campo con menu a tendina in cui selezionare il Gruppo (Ispettore Sanitario, Responsabile  

              Cofathec, etc.).

      ·         Richiedente: è un campo di ricerca libera dove inserire il nominativo del Richiedente. Se si conosce 

              esattamente il nominativo del richiedente lo si può inserire direttamente  altrimenti si può inserire una parte di

              esso e selezionare il match code  e si apre una maschera contente tutti i richiedenti che contengono il  

              valore inserito.

       Se invece si vuole visualizzare la lista completa dei richiedenti, occorre lasciare il campo vuoto e fare click direttamente sul match code .

      ·         Urgenza: è un campo con menu a tendina in cui selezionare l’ Urgenza (Max. Emergenza, Emergenza, etc.).   

      ·         Descrizione: è un campo di ricerca libera in cui inserire la descrizione da ricercare.

      ·         Tipo Manutenzione: è un campo con menu a tendina in cui selezionare il Tipo Manutenzione (Manutenzione

               su Richiesta, Pulizia Impianti, etc.).

               Se nel campo Tipo Manutenzione viene selezionata “Manutenzione Straordinaria” verranno visualizzati altri

               tre campi di ricerca:

1.       Tipo Intervento: è un campo con menu a tendina in cui selezionare il Tipo Intervento (Riq, etc.).

2.       Data Prevista Inizio: è un campo di ricerca dove inserire la data prevista di inizio intervento. Per inserire una data bisogna selezionare il pulsante  che ci aprirà una finestra contenete il calendario, selezionando un giorno in automatico verrà riportato sul campo Data Prevista Inizio.

3.       Data Prevista Fine: è un campo di ricerca dove inserire la data prevista di fine intervento. Per inserire una data bisogna selezionare il pulsante  che ci aprirà una finestra contenete il calendario, selezionando un giorno in automatico verrà riportato sul campo Data Prevista Fine.

 

 

L’inserimento di due o più campi di ricerca determina la selezione di file che soddisfano tutti i criteri inseriti (AND logico).

Selezionando il pulsante si rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

Selezionare il pulsante  per visualizzare, nel frame in basso, il risultato della ricerca in base ai criteri impostati.

 

Per Creare un file .Pdf bisogna selezionare una o più RdL. Le RdL si selezionano spuntando il check posta vicino alla RdL di interesse. Se si vogliono selezionare tutte le RdL basta selezionare il Pulsante  e in automatico tutte le RdL vengono selezionate. Se invece si vogliono deselezionare le RdL precedentemente selezionate basta selezionare il pulsante e in automatico tutte le RdL vengono deselezionate.

Una volta selezionate le RdL di interesse cliccare sul pulsante  per confermare le RdL selezionate e in basso apparirà   che ci indica quante RdL sono state selezionate. Una volta confermate le RdL selezionare il pulsante e in automatico crea il file pdf per quante RdL Selezionate. Per visualizzare i file creati selezionare il pulsante  e si aprirà la seguente maschera

 

Selezionando l’icona  verrà visualizzata la stampa .pdf creata nella pagina precedente.

Selezionando l’icona  si aprirà una finestra che ci permette di salvare il file che si trova sul server sulla nostra maschera in locale.

Selezionando l’icona  si potrà cancellare il record d’interesse.

Selezionando l’icona si visualizzerà  la maschera di dettaglio
