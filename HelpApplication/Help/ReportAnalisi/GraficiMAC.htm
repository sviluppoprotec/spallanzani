Grafici MAC

 

 

Selezionando la voce di menu Grafici MAC,  l’utente visualizza la seguente maschera che permette di produrre un Grafico per:

 

·       Data Da:   A: selezionare le date, attraverso il match code , per filtrare il periodo in cui sono state inserite le richieste.

Selezionare il tipo di Opzione per:

·       Data Di Richiesta:

·       Data Assegnazione:

·       Data Chiusura:

·       Tipologia Intervento:

·       RdL per Mese

·       RdL per Ditta

·       RdL per Servizio nei Mesi

·       RdL per Ditta nei Mesi

·       RdL pre Stato

·       RdL per Servizio

 

 

 

Selezionando il pulsante      per visualizzare, nel frame in basso, le RdL  che corrispondono ai criteri di generazione impostati.                           

 

Selezionare il pulsante       per rimuovere il contenuto dai campi valorizzati.

 

 

Nell’esempio di seguito riportato, sono state richieste le Rdl Per Mese, per il periodo dal 01/01/2005 al 18/11/2005.
