Navigazione Documenti

 

Selezionando la voce di menu Navigazione Documenti, l’utente visualizza la maschera Navigazione Documenti che consente di visualizzare, attraverso vari parametri di ricerca, la documentazione tecnico-normativa (Planimetrie, Foto, Schemi, Relazioni Tecniche) relativa ad un edificio oppure alle apparecchiature in esso presenti.

Questa maschera di ricerca è simile a quella vista in precedenza ed ha le medesime funzionalità, l’unica differenza è un nuovo campo nel quale inserire la descrizione del file che si sta cercando e il campo Nome file al posto del campo Standard.

La ricerca può avvenire, pertanto, nella stessa maniera vista in precedenza, con la possibilità di inserire direttamente il nome del file oppure la sua descrizione.

 

La ricerca può essere effettuata tramite i seguenti campi:

 

          Ricerca Edificio: è un campo di ricerca libera che consente di fare una ricerca per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  per aprire finestra con la lista dei Codici Edificio.

Il sistema visualizza, se ci sono, tutti gli edifici che contengono tale parola chiave. Dopo aver compilato il campo

               fare click sul pulsante  per aprire la finestra con la lista dei Codici Edificio.

               Il sistema visualizza, se ci sono, tutti gli Edifici che contengono tale parola chiave. Dopo aver selezionato il         

Codice dell’Edificio il sistema valorizza automaticamente il campo Codice Edificio.

         Codice Edificio (Obbligatorio): ): conoscendo esattamente il numero dell’Edificio/Complesso, o parte di esso, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una maschera che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “87” e selezionando il match code verranno presentati tutti gli edifici contenenti “8” e “7” del tipo “087”, “187”, “287”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

 

Se si vuole fare una ricerca più affinata, si possono compilare, oltre il campo Codice Edificio che è obbligatorio e quindi non può essere lasciato vuoto, anche i seguenti campi:

·         Nome File: è un campo di ricerca libera in cui inserire il nome o parte del nome del file da ricercare.

Ad esempio se si inserisce “PR” verranno elencati tutti i documenti che contengono “PR” nel proprio nome.

·          Descrizione File: è un campo sulla descrizione del file.

Per esempio digitando “Planimetria” verranno elencate tutti i documenti che contengono “Planimetria” nella descrizione.

·         Categoria: è un campo con menu a tendina con cui scegliere la categoria di file desiderato (Certificazione, Diagnosi Energetica, Relazione Tecnica, etc.).

·         Tipologia File: è un campo con menu a tendina con cui scegliere il tipo di file desiderato (Documento di testo, Foto, Planimetria, etc.).

L’inserimento di due o più campi di ricerca determina la selezione di file che soddisfano tutti i criteri inseriti (AND logico).

 

Selezionando il pulsante si rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

Selezionare il pulsante  per visualizzare, nel frame in basso, il risultato della ricerca in base ai criteri impostati. . Per ognuno di essi vengono date le seguenti informazioni di dettaglio:

 

-          Cod. Edifici

-          Servizio

-          Nome File

-          Tipo

-          Descrizione

 

 

Selezionando l’icona , si apre una finestra che mostra il documento corrispondente.

Facendo click, ad esempio, in corrispondenza del file d’interesse,  si apre la finestra con i dati generali dell’apparecchiatura  e se presente una foto della stessa.
