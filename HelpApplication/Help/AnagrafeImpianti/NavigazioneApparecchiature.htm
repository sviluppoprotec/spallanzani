Navigazione Apparecchiature

 

Selezionando la voce di menu Navigazione Apparecchiature, l’utente visualizza la maschera Navigazione Apparecchiature che permette, tramite diverse modalità di ricerca, di risalire alla scheda tecnica di un’apparecchiatura presente in un edificio.

 

 

·Codice Edificio: conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

·Ricerca Edificio: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  . Il sistema visualizza una Lista degli edifici, se ci sono, che contengono tale parola chiave.

·Piano: è possibile effettuare la ricerca selezionando il piano dell’edificio.

·Stanza: è possibile effettuare la ricerca attraverso il match code e la casella di testo. Il match code apre la finestra “Lista Stanze” con l’elenco delle Stanze disponibili. Il filtro di ricerca evidenzierà le apparecchiature relative alla Stanza selezionata.

·Servizio: è possibile effettuare la ricerca attraverso il match code e la casella di testo. Il match code apre la finestra “Lista Specializzazioni” con l’elenco dei Servizi disponibili. Il filtro di ricerca evidenzierà le apparecchiature con una particolare tipologia di servizio (Elettrico, Idrico, Sollevamento, etc.).

·Standard Apparecchiatura: attraverso il menu a tendina è possibile applicare un filtro di ricerca per evidenziare le apparecchiature appartenenti ad un determinato Standard (Compartimentazione, Utilizzazioni Elettriche, Impianto Idrosanitario, etc.).

·Codice Apparecchiatura: conoscendo esattamente il numero del Codice Apparecchiatura, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Apparecchiatura che contengono i valori inseriti interamente o in parte. Ad esempio, digitando “CIL” e selezionando il match code verranno presentate tutte le apparecchiature contenenti “CIL” (Corpi Illuminanti) nel proprio Codice Apparecchiatura (“FSL-G1_COP_F04-01.01.03_01”, “FSL-G1_COP_F04-01.01.03_02”, etc.). Se invece si vuole visualizzare la lista completa di tutte le apparecchiature, occorre lasciare il campo vuoto e fare click direttamente sul match code.

L’inserimento di due o più campi di ricerca determina la selezione di file che soddisfano tutti i criteri inseriti (AND logico)

Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto CANC della tastiera per i campi testo o selezionare la voce vuota nel menù a tendina.

 

Selezionare il pulsante  per visualizzare nel frame in basso, il risultato della ricerca in base ai criteri applicati.

La lista comprende il Codice Edificio, Codice Apparecchiatura, Standard Apparecchiatura ed il Servizio relativo a ciascuna apparecchiatura.

 


Facendo click sull’icona   dell’apparecchiatura d’interesse, si apre la finestra Scheda Apparecchiatura, con i relativi dettagli

 


Facendo click sull’icona (visualizza Richieste di Lavoro) dell’apparecchiatura d’interesse, si apre la finestra Navigazione  Apparecchiature
