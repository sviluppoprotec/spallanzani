Navigazione Certificati

Selezionando la voce di menu Navigazione Certificati, l’utente visualizza la maschera Navigazione Certificati che permette, tramite diverse modalità di ricerca, di risalire ai certificati legati ad un edificio o alle apparecchiature.

·         Codice Edificio: conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

·          Ricerca: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  . Il sistema visualizza una Lista degli edifici, se ci sono, che contengono tale parola chiave.

·         Nome File: è un campo di ricerca libera in cui inserire il nome o parte del nome del file da ricercare.

·          Descrizione File: è un campo sulla descrizione del file.

·          Tipo:  è un campo con menu a tendina in cui scegliere il tipo di file desiderato, riporterà infatti un elenco con tutti i certificati disponibili.

 

Selezionare il pulsante  per visualizzare, nel frame in basso, il risultato della ricerca in base ai criteri impostati. . Per ognuno di essi vengono date le seguenti informazioni di dettaglio:

-          Cod. Edifici

-          Servizio

-          Nome File

-          Tipo

-          Descrizione

 

Selezionando il pulsante  si rimuovere il contenuto dai campi valorizzati. Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

Selezionando l’icona , si apre una finestra che mostra il certificato corrispondente.
