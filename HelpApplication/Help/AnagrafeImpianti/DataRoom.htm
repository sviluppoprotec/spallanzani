Data Room

 

Selezionando la voce di menù Data Room, l’utente visualizza la maschera Data Room che permette, tramite diverse modalità di ricerca, di visualizzare l’apparecchiatura presente in un Edificio, distinto per Piano e per Stanza.

 

 

 

 

 

La ricerca può essere effettuata tramite i seguenti campi:

 

 

·          Codice Edificio: conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri/lettere inseriti. Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

·          Ricerca: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  . Il sistema visualizza una Lista degli edifici, se ci sono, che contengono tale parola chiave.

 

·          Piano: è un campo con menù a tendina con cui scegliere il Piano d’interesse

·          Stanza: selezionare  il match code  e all’apertura del PopUp, scegliere la stanza d’interesse.

·         Servizio (campo obbligatorio) : è un campo con menu a tendina con cui scegliere dall’elenco dei Servizi le apparecchiature con una particolare tipologia di servizio (Elettrico, Idrico, Sollevamento, etc.).

 

·         Standard Apparecchiatura: è un campo con menu a tendina con cui scegliere dall’elenco con una particolare apparecchiatura legata al Servizio precedentemente selezionato.

  • Codice Apparecchiatura: conoscendo esattamente il numero del Codice Apparecchiatura, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Apparecchiatura che contengono i valori inseriti interamente o in parte. Se invece si vuole visualizzare la lista completa di tutte le apparecchiature, occorre lasciare il campo vuoto e fare click direttamente sul match code.

Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.
