Funzioni

 

Selezionando la voce di menu Funzioni, l’utente visualizza la maschera Funzioni che permette, tramite diverse modalità di ricerca, di risalire all’elenco di tutte le Funzioni inserite.

 

 

 

La ricerca può essere effettuata attraverso i seguenti  campi:

·          Descrizione

·          Codice

I suddetti sono campi di ricerca libera che consentono di fare una ricerca per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino la Funzione d’interesse.

L’inserimento di due o più campi di ricerca determina la selezione di file che soddisfano tutti i criteri inseriti (AND logico).

Dopo aver compilato i campi fare click sul pulsante  per aprire la finestra con la lista delle Funzioni.

 

 

Per rimuovere il contenuto dai campi valorizzati Per ripulire il contenuto dei campi di ricerca fare click sul pulsante  .

Per analizzare la guida della funzione Funzioni fare click sul link GUIDA

 

 

 

Per  modificare o eliminare una Funzione  fare click sull’icona  posta accanto alla Funzione  di interesse.

 

 

Dopo aver apportato le modifiche desiderate  fare click sul tasto SALVA per dare avvio alla  modifica.

Per l’eliminazione fare click sul tasto ELIMINA, verrà richiesta una conferma all’eliminazione

Il tasto ANNULLA riporta all’elenco delle Funzioni inserite.

 

Selezionando il link  si apre la maschera per l’inserimento di una nuova Funzione.

 

L’inserimento di una nuova Funzione necessita l’inserimento dei campi:

·          Codice

·          Descrizione (campo obbligatorio)

·          Link File

·          Link File Help

Terminato l’inserimento dei dati relativi alla nuova Funzione, fare click sul pulsante SALVA.

Il tasto ANNULLA riporta alla maschera iniziale delle Funzioni.
