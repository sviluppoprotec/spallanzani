Cambio Password

 

Selezionando la voce di menu Cambio Password, l’utente visualizza la maschera Cambio Password che permette di modificare la password esistente.

La maschera visualizzata è la seguente:

 

 

 

 

 

Per cambiare la password bisogna compilare i seguenti campi:

·          Password Corrente: campo libero dove inserire la password che si ha.

·          Nuova Password: campo libero dove inserire la nuova password.

·          Conferma Password: campo libero dove inserire la nuova password per conferma.

Una volta compilati tutti i campi selezionare il pulsante  per rendere effettiva la modifica della password.
