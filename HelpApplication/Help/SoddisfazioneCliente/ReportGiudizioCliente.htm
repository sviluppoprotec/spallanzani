Report Giudizio Cliente

 

 

Selezionando la voce di menu Report Giudizio Cliente,  l’utente visualizza la maschera  Report Giudizio Cliente che permette di Esportare il file in formato Excel.

 

 

 

La ricerca può essere effettuata attraverso i seguenti  campi:

·          Cod. Edificio

·          Ricerca Edificio

·          Data Da – Data A

·          Servizio

·          Report Giudizio Cliente

·          Tipo Giudizio

Dopo aver compilato i campi fare click sul pulsante  per aprire il frame in basso contenente la lista  dei Giudizi del Cliente.

 

 

 

Per rimuovere il contenuto dai campi valorizzati Per ripulire il contenuto dei campi di ricerca fare click sul pulsante  .

Per analizzare la guida della funzione Report Giudizio Cliente  fare click sul link GUIDA.

 

 

 

Cliccando sul pulsante   l’utente ha la possibilità di aprire o Salvare il file prodotto, in formato Excel.
