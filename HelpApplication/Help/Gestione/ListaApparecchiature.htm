Anagrafe Apparecchiature

 

Selezionando la voce di menu Anagrafe Apparecchiature, l’utente visualizza la maschera Anagrafe Apparecchiature che permette, tramite diverse modalità di ricerca, di risalire all’elenco di tutte le Apparecchiature inserite.

 

 

La ricerca può essere effettuata attraverso i seguenti  campi:

·          Cod. Edificio (campo obbligatorio): conoscendo esattamente il numero/lettera dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri/lettera inseriti. Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

·          Ricerca Edificio: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  . Il sistema visualizza una Lista degli edifici, se ci sono, che contengono tale parola chiave.

·          Piano: è un campo con menù a tendina con cui scegliere il Piano d’interesse

·          Stanza: selezionare  il match code  e all’apertura del PopUp, scegliere la stanza d’interesse..

·          Servizio (campo obbligatorio) : è possibile effettuare la ricerca attraverso il match code e la casella di testo. Il match code apre la finestra “Lista Specializzazioni” con l’elenco dei Servizi disponibili. Il filtro di ricerca evidenzierà le apparecchiature con una particolare tipologia di servizio (Elettrico, Idrico, Sollevamento, etc.).

§          Standard Apparecchiatura: è un campo con menu a tendina con cui scegliere dall’elenco con una particolare apparecchiatura legata al Servizio precedentemente selezionato.

 

 

 

Dopo aver compilato i campi fare click sul pulsante  per aprire la PopUp (finestra) con la lista delle Apparecchiature

 

Per rimuovere il contenuto dai campi valorizzati Per ripulire il contenuto dei campi di ricerca fare click sul pulsante  .

Per analizzare la guida della funzione Anagrafe Apparecchiature fare click sul link GUIDA.

 

Per  modificare una apparecchiatura fare click sull’icona          posta accanto all’Apparecchiatura  di interesse.

 

Per  eliminare un apparecchiatura fare click sull’icona   posta accanto all’Apparecchiatura  di interesse.                                                                                   

 

Selezionando il link   si apre la maschera per l’inserimento di una nuova Apparecchiatura

                                                           

Per l’inserimento di una nuova Apparecchiatura è necessario compilare i campi:

·          Standard Apparecchiature: Standard a cui associare l’apparecchiatura

·          Data Inizio e Fine Validità: Data di inizio e fine contratto

·          QTA: Numero di apparecchiature presenti nel sito considerato

·          Condizione: Stato dell’Apparecchiatura

·          Piano

·          Stanza                                                                                  

·          Macro Competente: Entità superiore di cui fa parte l’apparecchiatura

·          Fornitore

·          Modello: Modello dell’apparecchiatura

·          Tipo: Tipo dell’Apparecchiatura

·          Dimesso: inserire flag se non più in uso

·          Immagine Allegata

·          Riferimento Planimetria

 

I campi obbligatori devono essere compilati per il corretto inserimento della nuova Apparecchiatura

Se non  si compilano questi campi campo e si clicca sul tasto SALVA verrà visualizzato un Alert di questo tipo
