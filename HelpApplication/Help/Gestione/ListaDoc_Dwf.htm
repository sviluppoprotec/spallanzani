Anagrafe Documenti

 

 

Selezionando la voce di menu Anagrafiche Documenti, l’utente visualizza la maschera Anagrafe Documenti che permette, tramite diverse modalità di ricerca, di risalire all’elenco di tutti i documenti inseriti.

 

 

Per avviare la ricerca è necessario selezionare un codice edificio.

conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri inseriti.

 

 

 

Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

Altrimenti è presente un campo di ricerca libera in cui inserire il nome dell’Edificio.

Ad esempio digitando “via Roma” la ricerca produrrà l’elenco delle apparecchiature di tutti quegli edifici che contengono “LS” (Liceo Scientifico) nella Denominazione.

L’inserimento di due o più campi di ricerca determina la selezione di file che soddisfano tutti i criteri inseriti (AND logico).

Dopo aver selezionato il Codice dell’Edificio il sistema è mostra tutti i documenti associati.

 

 

Selezionare il pulsante  per rimuovere il contenuto del campo valorizzato.

Per inserire un nuovo documento fare click sul link

 

 

L’inserimento di un nuovo documento nell’anagrafico necessita l’inserimento dei campi:

·          Servizio         (campo obbligatorio)  dove indicare il Servizio di cui fa parte ;

·          Tipologia del File (campo obbligatorio) dove indicare il tipo del file che si sta allegando

·          Categoria Generale

·          Categoria

·          Tipologia del Documento

La tipologia di Documento è una variante che modifica i successivi campi da inserire.

Per allegare il file e’ necessario cliccare su SFOGLIA e scegliere nel proprio disco il file.

Per completare l’inserimento fare click sul tasto SALVA, in questo modo il programma associa un  nome file automaticamente.

.

Il tasto INDIETRO  interrompe l’operazione, cancella gli eventuali dati inseriti e riporta alla pagina iniziale “Anagrafe Documenti”

 

Per eliminare un Documento gia inserto fare click sul tasto ELIMINA accanto al tipo documento di interesse. Prima di procedere all’effettiva eliminazione verrà richiesta una conferma.

Selezionando l’icona       in corrispondenza del documento  da esaminare si apre la finestra per la visualizzazione del dettaglio, tale visualizzazione non permette di modificare le caratteristiche del documento.

 

 

Selezionando il tasto NUOVO FILE si dà la possibile di inserimento nuovo documento con le stesse caratteristiche di quello selezionato.

Il nome del file viene generato automaticamente dal programma.

Il tasto INDIETRO  interrompe l’operazione, cancella gli eventuali dati inseriti e riporta alla pagina iniziale “Anagrafe Documenti”
