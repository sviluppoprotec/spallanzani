Lettura contatori

 

Selezionando la voce di menu Lettura contatori, l’utente visualizza la maschera Lettura Contatori che permette, tramite diverse modalità di ricerca, di risalire all’elenco delle lettura inserite.

 

 

La ricerca può essere effettuata attraverso i seguenti  campi:

·  Codice Edificio: conoscendo esattamente il nome dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i caratteri inseriti. Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

·  Ricerca Edificio: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave. L’utente ha la possibilità di inserire dati parziali, (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  . Il sistema visualizza una Lista degli edifici, se ci sono, che contengono tale parola chiave.

·       Data Lettura Da   A: selezionare le date, attraverso il match code , per filtrare il periodo in cui sono state effettuate le letture.

·        Piano: è possibile effettuare la ricerca selezionando il piano dell’edificio.

·        Stanza: è possibile effettuare la ricerca attraverso il match code e la casella di testo. Il match code apre la finestra “Lista Stanze” con l’elenco delle Stanze disponibili. Il filtro di ricerca evidenzierà le apparecchiature per cui è prevista una lettura relative alla Stanza selezionata.

·        Servizio: nel menu a tendina è presente l’elenco dei Servizi disponibili. Il filtro di ricerca visualizzerà  le apparecchiature per cui è prevista una lettura con una particolare tipologia di servizio (Elettrico, Idrico, Sollevamento, etc.).

·        Standard Apparecchiatura: attraverso il menu a tendina è possibile applicare un filtro di ricerca per evidenziare le apparecchiature appartenenti ad un determinato Standard (Compartimentazione, Utilizzazioni Elettriche, Impianto Idrosanitario, etc.).

·        Codice Apparecchiatura: conoscendo esattamente il numero del Codice Apparecchiatura, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Apparecchiatura che contengono i valori inseriti interamente o in parte. Ad esempio, digitando “CIL” e selezionando il match code verranno presentate tutte le apparecchiature contenenti “CIL” (Corpi Illuminanti) nel proprio Codice Apparecchiatura (“CT-G1_COP_F04-01.01.03_01”, “CT-G1_COP_F04-01.01.03_02”, etc.). Se invece si vuole visualizzare la lista completa di tutte le apparecchiature, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

L’inserimento di due o più campi di ricerca determina la selezione di file che soddisfano tutti i criteri inseriti (AND logico)

Dopo aver compilato i campi fare click sul pulsante  per aprire la lista delle letture.

 Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto CANC della tastiera per i campi testo o selezionare la voce vuota nel menù a tendina.

Selezionando l’icona   in corrispondenza della lettura che si vuole esaminare verrà visualizzata la maschera di modifica

dove sarà possibile apportare i cambiamenti desiderati senza però la possibilità di cambiare l’apparecchiatura, nella parte “Filtra Apparecchiatura” sono riportati i dettagli dell’apparecchiatura per cui è stata effettuata la lettura.

Selezionare il pulsante  per confermare le modifiche apportate, selezionare il pulsante  per eliminare la lettura per tornare invece alla pagina di ricerca selezionare il pulsante .

 

Selezionando il link   si apre la maschera per l’inserimento di una nuova lettura.

 

 

La prima parte della maschera, Filtra Apparecchiatura

può essere utilizzata per filtrare l’apparecchiatura per la quale si desidera inserire la lettura.

 

 

La seconda parte Salva Lettura

serve per l’inserimento della lettura i campi necessari per effettuare questa operazione sono:

·        Codice Apparecchiatura: conoscendo esattamente il numero del Codice Apparecchiatura, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Apparecchiatura che contengono i valori inseriti interamente o in parte,. Inoltre se sono stati valorizzati alcuni, o tutti, i filtri di ricerca della sezione “Filtra Apparecchiatura”  verranno usati per ricercare l’apparecchiatura.

·        Data Ora Lettura

·        Valore Lettura

·        Addetto

·        Energia

·        Nota

Il tasto  interrompe l’operazione e cancella gli eventuali dati inseriti riportando l’utente alla pagina di ricerca.
