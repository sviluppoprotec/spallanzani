Scarica File Pdf

 

 

Selezionando la voce di menu Gestione Stampe Rapportini  MP, l’utente visualizza la maschera Download Stampe Rapporti (Immagine 1) che permette di visualizzare tutti i file precedentemente creati e depositati sul server, ordinati in base della data di creazione.

 

 

Immagine 1

 

Le quattro funzionalita di questa pagina sono presentati con seguenti bottoni:

 

1)       - visualizza file .pdf nella pagina successiva (Immagine 2)

 

 

 

Immagine 2

 

1)       - download del file .pdf archiviato con Winzip dal server al client

2)       - elimina file .pdf dal server

3)       - mostra tutti i criteri di ricerca che hanno generato il file (Immagine 3)

 

Immagine 3

 

Cliccando sul bottone  vengono eliminati tutti i file presenti nel server.

Ciccando sul bottone  si torna alla pagina iniziale.
