Schede Apparecchiatura

 

 

Selezionando la voce di menu Schede Apparecchiatura, l’utente visualizza la maschera Schede Apparecchiatura  che permette, tramite diverse modalità di ricerca, di risalire alla scheda tecnica di un’apparecchiatura presente in un edificio.

 

 

 

·         Codice Edificio (Obbligatorio): conoscendo esattamente il numero od il nome dell’Edificio/Complesso, o parte di esso, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una maschera che mostra solo quei Codici Edificio che contengono i numeri/lettere inserite.. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

Se si vuole fare una ricerca più affinata, si possono compilare, oltre il campo Codice Edificio che è obbligatorio e quindi non può essere lasciato vuoto, anche i seguenti campi:

·         Piano:  è un campo con menu a tendina con cui scegliere il Piano d’interesse.

·         Stanza:  conoscendo esattamente il numero della stanza, o parte di esso, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una maschera che mostra solo le stanze che contengono i numeri/lettere inserite.. Se invece si vuole visualizzare la lista completa delle stanze, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

·         Servizio (campo obbligatorio) : è un campo con menu a tendina con cui scegliere dall’elenco dei Servizi le apparecchiature con una particolare tipologia di servizio (Elettrico, Idrico, Sollevamento, etc.).

 

·         Standard Apparecchiatura: è un campo con menu a tendina con cui scegliere dall’elenco con una particolare apparecchiatura legata al Servizio precedentemente selezionato.

·         Cod. Apparecchiatura: conoscendo esattamente il codice dell’apparecchiatura, o parte di esso, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una maschera che mostra solo le apparecchiature che contengono i numeri/lettere inserite.. Se invece si vuole visualizzare la lista completa delle apparecchiature, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Dopo aver impostato i criteri sopraindicati, cliccare sul tasto  e verrà visualizzata la maschera che darà la possibilità di selezionare una o più apparecchiature, di generare Report Html, generare Report Pdf, ecc.  tramite gli appositi sottostanti pulsanti
