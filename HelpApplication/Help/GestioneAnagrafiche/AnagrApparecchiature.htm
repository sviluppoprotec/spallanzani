Anagrafe Apparecchiature

 

Selezionando la voce di menu Anagrafe Apparecchiature, l’utente visualizza la maschera Anagrafe Apparecchiature che permette, tramite diverse modalità di ricerca, di risalire all’elenco di tutte le Apparecchiature inserite.

 

 

La ricerca può essere effettuata attraverso i seguenti  campi:

·          Cod. Edificio (campo obbligatorio): conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

·          Ricerca Edificio: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  . Il sistema visualizza una Lista degli edifici, se ci sono, che contengono tale parola chiave.

·          Servizio (campo obbligatorio) : è possibile effettuare la ricerca attraverso il match code e la casella di testo. Il match code apre la finestra “Lista Specializzazioni” con l’elenco dei Servizi disponibili. Il filtro di ricerca evidenzierà le apparecchiature con una particolare tipologia di servizio (Elettrico, Idrico, Sollevamento, etc.).

 

Dopo aver compilato i campi fare click sul pulsante per aprire la PopUp (finestra) con la lista delle Apparecchiature

 

Per rimuovere il contenuto dai campi di ricerca valorizzati  fare click sul pulsante .

Per analizzare la guida della funzione Anagrafe Apparecchiature fare click sul link GUIDA.

 

Per  modificare una apparecchiatura fare click sull’icona posta accanto all’Apparecchiatura  di interesse.

 

 

Per  eliminare un apparecchiatura fare click sull’icona   posta accanto all’Apparecchiatura  di interesse.      

Selezionando il link si apre la maschera per l’inserimento di una nuova Apparecchiatura

                                                          

L’inserimento di una nuova Apparecchiatura necessita l’inserimento dei campi:

·          Standard Apparecchiature

·          Data Inizio e Fine Validità

·          QTA

·          Condizione

·          Piano

·          Stanza                                           descrizione campi                                         

·          Macro Competente

·          Fornitore

·          Modello

·          Tipo

·          Dimesso

·          Immagine Allegata

·          Riferimento Planimetria
