Gestione Documenti Manutenzione Programmata

 

Selezionando la voce di menu Gestione Documenti Manutenzione Programmata, l’utente visualizza la seguente maschera che permette di effettuare il download dei piani di manutenzione programmata.

 

 

Per effettuare la ricerca l'utente può utilizzare come filtri uno o più campi visualizzati all'interno della maschera. L’inserimento di più campi di ricerca determina la selezione di piani di manutenzione che soddisfano tutti i criteri inseriti (AND logico).

I campi visualizzati sono:

 

·          Edificio: conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una Pop Up che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “37” e selezionando il match code verranno presentati tutti gli edifici contenenti “3” e “7” nel proprio Codice Edificio, del tipo “037”, “337”, “370”, etc. e/o nel numero civico dell’indirizzo. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code corrispondente.

 

·          Nome Documento: è un campo di ricerca libera in cui inserire in parte o completamente il nome del documento  da cercare.

·          Stato: selezionare un valore attraverso il match code per indicare lo stato del piano.

 

·          Data invio e Data Inserimento: selezionare le date, attraverso il match code , per filtrare per la data in cui sono state spediti i piani di manutenzione. Si apre la Pop Up seguente:

 

 

 

 

 

 

 

 

 

 

 

 

 

 

·          Tipo di documento: selezionare un valore attraverso il match code per indicare il tipo di piano.

 

·          Anno: selezionare un valore attraverso il match code per indicare l’anno.

 


Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

 

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

 

 

 

Selezionare il tasto  per visualizzare, nel frame in basso, tutte le richieste che soddisfano i criteri impostati. Per ognuna di esse vengono date le seguenti informazioni di dettaglio:

 

- Download

- Edificio

- Nome

- Documento

- Stato

- Data Invio

- Data Inserimento

- Convalida

 

Selezionando l’icona   corrispondente alla descrizione del piano di manutenzione, l’utente può effettuare il download dell’ relativo piano.

Nell’ultima colonna “Convalida” si ha la possibilità di convalidare un Piano Proposto ad Acettato o un Piano Accettato in Eseguito, tale funzionalità

permette direttamente di creare nuove richieste e ordini in caso di Accettazione e di chiuderle in caso di Esecuzione.

Le date di accettazione e di chiusura saranno quelle dell’ultimo piano inserito. Oltre al link nella colonna di convalida è presente una csella di spunta

La quale se valorizzato il flag Invia email vengono inviati hai

destinatari preposi i nuovi piani rigenerati.
