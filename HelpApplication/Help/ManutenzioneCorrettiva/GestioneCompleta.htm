Completamento RdL Correttiva

 

Selezionando la voce di menu Gestione RdL Correttiva, si visualizza la maschera Gestione RdL Correttiva, che consente di visualizzare le Richieste di Lavoro Correttive secondo diversi criteri. Gli utenti autorizzati che accedono a questa maschera, possono solo visualizzare le richieste secondo diversi criteri, ma non può effettuare alcuna operazione su di esse.

 

 

La pagina consente di ricercare le Richieste di Lavoro precedentemente inserite ed in attesa di completamento o modifica.

Per ricercare le Richieste di Lavoro l'utente può utilizzare uno o più campi visualizzati all'interno della pagina come filtri per la ricerca.

 

I campi visualizzati sono:

·          Edificio: conoscendo esattamente il numero dell’Edificio/Complesso, lo si può digitare direttamente nel campo.   Se si seleziona il match code  dopo aver compilato il campo con dati parziali, si apre una Pop Up che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “10” e selezionando il match code verranno presentati tutti gli edifici contenenti “1” e “0” nel proprio Codice Edificio, del tipo “010”, “110”, “210”, etc. e/o nel numero civico dell’indirizzo. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code corrispondente.

       Ricerca: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave.  

       L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice 

       dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul  parola

       pulsante  . Il sistema visualizza una Lista degli edifici, se ci sono, che contengono  

       tale chiave.

·          Richiesta: è un campo di ricerca libera in cui inserire il numero della Richiesta da ricercare.

·          Data Da:       Data A: selezionare le date, attraverso il match code , per filtrare il periodo in cui è stata inserita la richiesta.

·          Ordine di Lavoro: è un campo di ricerca libera in cui inserire il numero dell’Ordine da ricercare.

·          Richiedente: è un campo di ricerca libera in cui inserire il nome della persona che ha richiesto l’intervento.

       Se si seleziona il match code  dopo aver compilato il campo con dati parziali, si apre una Pop Up che

       mostra tutti i Richiedenti che contengono il valore inserito. Ad esempio digitando “maria” e selezionando il 

       match code verranno presentati tutti i Richiedenti che contengono quella parola quindi “maria”, “marianna”  

       etc.

·          Operatore: è un campo di ricerca libera in cui inserire il nome dell’Operatore da ricercare.

·          Addetto: è un campo di ricerca libera in cui inserire il nome dell’ addetto.

       Se si seleziona il match code  dopo aver compilato il campo con dati parziali, si apre una Pop Up che

       mostra tutti gli addetti che contengono il valore inserito. Ad esempio digitando “b” e selezionando il 

       match code verranno presentati tutti gli Addetti che contengono quella parola.

·          Ditta: selezionare un valore attraverso l’elenco a  tendina per filtrare  la Ditta di competenza dell’intervento.

·          Gruppo: selezionare un valore attraverso l’elenco a tendina per filtrare il gruppo di appartenenza del Richiedente .

·          Descrizione: è un campo in cui inserire una parola chiave della descrizione del problema del componente interessato e/o dell’ubicazione.

·          Urgenza: selezionare un valore attraverso l’elenco a tendina per filtrare il tipo di urgenza della Richiesta di Lavoro .

·          Servizio: selezionare un valore attraverso l’elenco a tendina per filtrare i Servizi disponibili.

·          Tipo Manutenzione: selezionare un valore dal menù a tendina per filtrare il tipo manutenzione (straordinaria o su richiesta).

      

L’inserimento di più campi di ricerca determina la selezione di ordini che soddisfano tutti i criteri inseriti (AND logico).

 

Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o scegliere la voce vuota nei menu a tendina.

  

Selezionare il pulsante  per visualizzare, nel frame in basso, l'elenco delle Richieste di Lavoro  che corrispondono ai criteri di ricerca impostati. Per ognuna di esse vengono date le seguenti informazioni di dettaglio:

 

·       Edificio

·       Indirizzo

·       OdL

·       Data Emissione

·       Ditta

·       RdL

·       Data Creazione

·       Stato

·       Priorità

·       Data Pianificata

·       Annotazioni Aggiuntive

·       Tipo Manutenzione

 

 

Selezionando l’icona  corrispondente alla RdL da completare o modificare, l'utente visualizza la pagina Completa Richiesta di lavoro.

 

La sezione relativa alla Creazione Richiesta di Lavoro mostra tutte le informazioni della Richiesta.

 

 

 

La sezione relativa all’Emissione Ordine di Lavoro permette di modificare i seguenti campi:

      ·       Ditta Esecutrice: selezionare un valore attraverso il menù a tendina per assegnare la Ditta che eseguirà   

              l’intervento.

     ·       Addetto: selezionare un valore attraverso il menù a tendina per inserire il nominativo della persona addetta

             all’intervento.

     ·       Urgenza: assegnare una priorità all’intervento scegliendo il valore dal menu a tendina.

     ·       Data Pianificata: selezionare la data di pianificazione dell’intervento tramite il match code.

     ·       Orario Pianificato: selezionare un valore attraverso l’elenco a tendina, rispettando gli standard di formato  

             previsti (hh:mm).

 

 

 

 

La sezione relativa al Completamento Ordine di Lavoro permette di concludere l’Ordine valorizzando i seguenti campi:

 

·       Stato della Richiesta di Lavoro: selezionare un valore attraverso il menu a tendina.

·       Data Inizio Lavori: selezionare una Data Inizio Lavori attraverso il match code.

·       Ora Inizio: selezionare un valore attraverso i pulsanti di scorrimento per le ore e i minuti.

      ·       Data Fine Lavori: selezionare una Data Fine Lavori attraverso il match code.

·       Ora Fine Lavori: selezionare un valore attraverso i pulsanti di scorrimento per le ore e i minuti.

·       Annotazioni Materiale Utilizzato: campo di testo in cui annotare eventuali osservazioni sui materiali adoperati.

 

 

 

Selezionare il pulsante  per confermare l’emissione dell’Ordine di Lavoro.

 

Selezionare il pulsante  per visualizzare il Rapporto Tecnico di Intervento

 

Selezionando il pulsante  visualizzare il Rapporto Tecnico di Intervento in formato Pdf.

Selezionare il pulsante   della maschera principale per chiudere la finestra e sospendere lo stato di approvazione della Richiesta di Lavoro.

Selezionando il pulsante  verrà aperta la maschera in Pop Up per inserire un nuovo sollecito.

 

L'utente può stampare la scheda, contenente tutti i dettagli sull'intervento, per farla recapitare all’addetto che provvederà ad aggiungere note, firme e dettagli a lavoro ultimato.
