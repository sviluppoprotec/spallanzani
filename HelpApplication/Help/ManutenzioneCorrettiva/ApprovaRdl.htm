Approva ed Emetti Rdl

 

Selezionando la voce di menu Approva ed Emetti RdL l’utente semplice visualizza la maschera Approva ed Emetti Richieste di Lavoro, che permette di visualizzare lo stato delle richieste inserite a sistema e approvarle, sospenderle o rifiutarle. L’approvazione della richiesta e l’emissione del relativo ordine di lavoro sono, pertanto, possibili soltanto accedendo al sistema come utente amministratore.

 

 

 

Si possono compilare i seguenti campi:

 

·          Richiesta: è un campo di ricerca libera in cui inserire il numero della richiesta da approvare/emettere.

·          Operatore: è un campo di ricerca libera in cui inserire il nome dell’operatore che ha inserito l’intervento.

·          Gruppo: selezionare un valore attraverso l’elenco a tendina per filtrare il gruppo di appartenenza dell’operatore

·          Data Da:               A: selezionare le date, attraverso il match code , per filtrare il periodo in cui sono state inserite le richieste. Si apre la PopUp seguente:

 

 

 

 

 

 

 

 

 

 

 

 

 

·          Urgenza: è un campo con menu a tendina per indicare l’urgenza dell’intervento, all’operatore che andrà a visionare le richieste.

·          Validazione: è un campo con menù a tendina con cui scegliere la Validazione per Tutte le Richieste, Richieste non ancora validate, Richieste validate COLLABORATORE e Richieste validate DL.

·          Servizio: è un campo con menu a tendina con cui scegliere la categoria desiderata (Centrali Termiche, Sanitari, Scale, etc.).

·          Richiedente: è un campo di ricerca libera in cui inserire il nome della persona che ha richiesto l’intervento.

·          Edificio: conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “37” e selezionando il match code verranno presentati tutti gli edifici contenenti “3” e “7” nel proprio Codice Edificio, del tipo “037”, “337”, “370”, etc. e/o nel numero civico dell’indirizzo. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code corrispondente.

·           Ricerca: è un campo di ricerca libera che consente di fare una ricerca dell’edificio per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  .

Il sistema visualizza una Lista degli edifici, se ci sono, che contengono tale parola chiave.

 

 

 

L’inserimento di due o più campi di ricerca determina la selezione delle Richieste di Lavoro che soddisfano tutti i criteri inseriti (AND logico).

 

Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

 

Selezionare il link Guida per accedere all’Help On Line relativo alla form.

 

Selezionare il tasto  per visualizzare, nel frame in basso, tutte le richieste che soddisfano i criteri impostati. Per ognuna di esse vengono date le seguenti informazioni di dettaglio:

 

-          Cod. Edificio

-          Indirizzo

-          N. RdL

-          Data RdL

-          Stato

-          Servizio

-          Descrizione

-          Urgenza

-          Validazione

-          Operatore

-          Richiedente

-         
Gruppo

 

 

Selezionando l’icona  corrispondente alla descrizione della RdL, l’utente visualizza la pagina per effettuare l’approvazione della RdL in esame

 

 

Per procedere con l’approvazione della richiesta l’utente può valorizzare i seguenti campi:

·          Ditta Esecutrice: (obbligatorio) selezionare un valore attraverso il menu a tendina

·          Addetto: (obbligatorio) selezionare il nominativo della persona addetta all’intervento, tale menù sarà caricato automaticamente con gli addetti associati alla ditta scelta.

·          Urgenza: è un campo con menu a tendina per indicare l’urgenza dell’intervento, all’operatore che andrà a visionare le richieste. 

·          Data Pianificata: selezionare le date, attraverso il match code  . Si apre il PopUp seguente:

 

 

 

 

 

 

 

 

 

 

 

 

 

·          Orario Pianificato: è un campo con menu a tendina che permette di selezionare un determinato orario

 

 

 

 

 

 

Selezionare il pulsante  per rifiutare l’approvazione della Richiesta di Lavoro.

Selezionare il pulsante  per terminare l’operazione e tornare alla pagina iniziale.

Selezionare il pulsante  per sospendere lo stato di approvazione della Richiesta di Lavoro.

 

Selezionare il pulsante  per procedere con l’approvazione della Richiesta di Lavoro e visualizzare Buono di Lavoro.
