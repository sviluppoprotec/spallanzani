Elimina RdL

 

Selezionando la voce di menu Elimina RdL, l’utente visualizza la maschera Sfoglia RdL da eliminare che consente di visualizzare, attraverso vari parametri di ricerca, le RdL esistenti, per poi poterle eliminare.

 

 

La ricerca può essere effettuata tramite i seguenti campi:

 

·         Codice Edificio: conoscendo esattamente il nome dell’Edificio, o parte di esso, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una maschera che mostra solo quei Codici Edificio che contengono i caratteri inseriti. Ad esempio, digitando “FSL” e selezionando il match code verranno presentati tutti gli edifici contenenti “FSL”. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

 

 

·        Ricerca Edificio: è un campo di ricerca libera che consente di fare una ricerca per parola chiave. L’utente ha la possibilità di inserire dati parziali (almeno due caratteri), che individuino il Codice dell’Edificio, il Nome, la Descrizione, l’Indirizzo, la Ditta. Dopo aver compilato il campo fare click sul pulsante  per aprire finestra con la lista dei Codici Edificio.

Il sistema visualizza, se ci sono, tutti gli edifici che contengono tale parola chiave. Dopo aver compilato il campo

               fare click sul pulsante  per aprire la finestra con la lista dei Codici Edificio.

               Il sistema visualizza, se ci sono, tutti gli Edifici che contengono tale parola chiave. Dopo aver selezionato il         

Codice dell’Edificio il sistema valorizza automaticamente il campo Codice Edificio.

 

 

 

Se si vuole fare una ricerca più affinata, si possono compilare, oltre il campo Codice Edificio anche i seguenti campi:

·         Richiesta di Lavoro: è un campo di ricerca libera in cui inserire il numero della richiesta da cercare.

·        Addetto: selezionare l’addetto attraverso il match code ..

     ·         Data Da: selezionare la data, attraverso il match code , per filtrare il periodo in cui sono state inserite le 

              richieste.

      ·         Data A: selezionare la data, attraverso il match code , per filtrare il periodo in cui sono state inserite le 

              richieste.

      ·         Ordine di Lavoro: è un campo di ricerca libera in cui inserire il numero della OdL da cercare.

    ·         Stato Richiesta: selezionare un valore attraverso l’elenco a tendina per filtrare lo stato delle richieste da  

              approvare/emettere.

     ·         Servizio: selezionare un valore attraverso l’elenco a tendina per filtrare il tipo Servizio.

     ·        Gruppo: selezionare un valore attraverso l’elenco a tendina per filtrare il gruppo di appartenenza del 

              Richiedente .

     ·         Richiedente: indicare il nome della persona che richiede l’intervento. Se si conosce il nome lo si può  

              scrivere  direttamente nel campo, oppure scrivere una parte del nome e selezionare il pulsante match code

              , per filtrare tutti i richiedenti che il loro nominativo contenga i caratteri inseriti.

     ·     Urgenza: selezionare un valore attraverso l’elenco a tendina per filtrale la priorità della OdL (Emergenza,

              Critico,Ordinario ecc….).

 

      ·      Descrizione: è un campo in cui inserire una parola chiave della descrizione del problema del componente  

              interessato e/o dell’ubicazione.

 

L’inserimento di due o più campi di ricerca determina la selezione di file che soddisfano tutti i criteri inseriti (AND logico).

 

Selezionando il pulsante  si rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto Canc della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

Selezionare il pulsante  per visualizzare, nel frame in basso, il risultato della ricerca in base ai criteri impostati. . Per ognuno di essi vengono date le seguenti informazioni di dettaglio:

 

-          Indirizzo

-          Odl

-          Data Emissione

-          Ditta

-          RdL

-          Data Creazione

-          Stato richiesta

-          Priorità

-          Data Pianificata

-          Data Fine Lavori

-          Descrizione

-          Tipo Man.

 

 

Selezionando l’icona , si apre una finestra che mostra la Richiesta di Lavoro che si è selezionato.

 

Selezionando il pulsante  si elimina la Rdl selezionata.

Selezionando il pulsante  si torna alla maschera di ricerca.

 

Selezionando l’icona  si può cancellare automaticamente da qui la RdL selezionata.
