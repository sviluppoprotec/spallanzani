Registro spedizioni SGA DIE

 

Selezionando la voce di menu Registro spedizioni SGA DIE, l’utente visualizza la seguente maschera che permette di visualizzare le SGA DIE già inviate.

 

 

Per effettuare la ricerca l'utente può utilizzare come filtri uno o più campi visualizzati all'interno della maschera. L’inserimento di più campi di ricerca determina la selezione di SGA DIE che soddisfano tutti i criteri inseriti (AND logico).

I campi visualizzati sono:

 

·          Edificio: conoscendo esattamente il numero dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una Pop Up che mostra solo quei Codici Edificio che contengono i numeri inseriti. Ad esempio, digitando “37” e selezionando il match code verranno presentati tutti gli edifici contenenti “3” e “7” nel proprio Codice Edificio, del tipo “037”, “337”, “370”, etc. e/o nel numero civico dell’indirizzo. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code corrispondente.

 

·          Numero Richiesta: è un campo di ricerca libera in cui inserire il numero della richiesta da cercare ( RDL).

 

·          Codice: è un campo di ricerca libera in cui inserire completamente il nome dell’ SGA DIE da cercare.

 

·          Nome Documento: è un campo di ricerca libera in cui inserire in parte o completamente il nome del documento  da cercare (SGA DIE).

 

·          Tipo Documento: selezionare un valore attraverso il match code per indicare la tipologia dell’documento (SGA DIE).

·          Data invio: selezionare le date, attraverso il match code , per filtrare per la data in cui sono state spedite le SGA DIE. Si apre la Pop Up seguente:

 

 

 

 

 

 

 

 

 

 

 

 

·          Utente: è un campo di ricerca libera in cui inserire il nome dell’ ultimo utente che ha inviato l’ SGA la DIE.

 


Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

 

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto “Canc” della tastiera per i campi testo o selezionare la voce vuota nel menu a tendina.

 

 

 

Selezionare il tasto  per visualizzare, nel frame in basso, tutte le richieste che soddisfano i criteri impostati. Per ognuna di esse vengono date le seguenti informazioni di dettaglio:

 

- Download   

- Re Invia       

- Cod. Edificio         

- N. RdL        

- Codice        

- Nome Documento 

- Tipo Doc.   

- Data Invio   

- Utente

 

 

 

Selezionando l’icona  corrispondente alla descrizione della SGA, DIE , l’utente visualizza un pop up dove  visionare i destinatari prefissati di questa SGA, DIE

 

Selezionando l’icona   corrispondente alla descrizione della SGA, DIE, l’utente re invia la SGA, DIE a i destinatari prefissati.

 

Selezionando l’icona   corrispondente alla descrizione della SGA o DIE, l’utente può effettuare il download della relativa SGA o DIE.

 

 

Selezionando l’icona nella colonna N. Rdl corrispondente alla descrizione della SGA, DIE, l’utente aprirà la relativa RDL.
