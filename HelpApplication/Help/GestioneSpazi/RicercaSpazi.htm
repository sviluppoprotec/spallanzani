Ricerca Spazi

 

 

Selezionando la voce di menu Ricerca Spazi,  l’utente visualizza la seguente maschera che permette la ricerca per Edificio, Piano, Stanza, Categoria, Destinazione d’Uso, Reparto e Mq2,  degli Spazi espressi in metri quadrati.

 

 

 

 

 

La ricerca può essere effettuata tramite i seguenti campi:

 

·          Codice Edificio: conoscendo esattamente il numero od il nome dell’Edificio, lo si può digitare direttamente nel campo. Se si seleziona il match code  dopo aver compilato il campo, si apre una PopUp che mostra solo quei Codici Edificio che contengono i numeri/lettere inseriti. Ad esempio, digitando “06” e selezionando il match code verranno presentati tutti gli edifici contenenti “0” e “6” del tipo “006”, “060”, “067”, etc. Se invece si vuole visualizzare la lista completa degli edifici, occorre lasciare il campo vuoto e fare click direttamente sul match code.

·          Piano: è possibile selezionare un piano specifico attraverso il menù a tendina

·          Stanza: attraverso il match code  si potrà selezionare una stanza in particolare

·          Categoria: è possibile selezionare una Categoria specifica attraverso il menù a tendina

·          Destinazione d’uso: attraverso il match code  si potrà selezionare la Destinazione d’uso in particolare

·          Reparto: attraverso il match code  si potrà selezionare il Reparto in particolare

·          Mq2: è possibile selezionare un valore (=, >, <) attraverso il menù a tendina

 

 

 

 

 

 

E’ inoltre possibile, inserendo una flag nell’apposita casella, visualizzare nella lista che si andrà a produrre, il campo d’interesse, come da esempio riportato:

 

 

 Inserendo il flag nella casella  avremo una lista che comprenderà tutte le informazioni relative ai campi (Codice Edificio, Piano, Stanza, ecc.)

 

 

 

Selezionare il pulsante  per rimuovere il contenuto dai campi valorizzati.

Per rimuovere invece, il valore dal singolo campo, utilizzare il tasto CANC della tastiera per i campi testo o selezionare la voce vuota nel menù a tendina.

 

Selezionare il pulsante  per visualizzare nel frame in basso, il risultato della ricerca in base ai criteri applicati.
